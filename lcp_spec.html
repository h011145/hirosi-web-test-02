LCP: 魔法のプロトコル

"エディタを使わずに、世界を創る方法"

1. 空間をひらく（mkdir）

何もないところに、プログラムが住むための「部屋」を作ります。 魔法の呪文 `mkdir -p` を唱えれば、一瞬で複雑な間取りの秘密基地ができあがります。

2. 魂をそそぐ（cat）

作った部屋の中に、AIが書いた「知能」を流し込みます。 `cat` コマンドは、AIの言葉をそのまま現実のファイルへと定着させる魔法です。

3. 命をふきこむ（chmod / python）

最後に、そのプログラムに「動け！」と命令します。 エディタを開いてカチカチ直す必要はありません。全ては一撃のコマンドで完結します。

// LCPの真理 //
人間は「何をしたいか」を言い、
AIは「どうやるか（コマンド）」を書き、
Linuxは「それ」を現実に変える。
