# LCP MISSION CONTROL - README

## ようこそ、リナックス研究会公式ターミナルへ
このページは、AIとLinuxを直結させ、ソフトウェア開発のプロセスを根底から破壊（クラック）するためのプロトコル **LCP (Linux Communication Protocol)** の司令室です。

---

## LCPとは？ - 魔法の3ステップ
LCPは、AIに「魔法の呪文」を唱えさせ、あなたのコンピュータの中に一瞬で「自分だけの秘密基地（ソフトウェア）」を作り出す技術です。

1.  **【STEP 1: 指令の抽出】**
    *   `h_chat.html` (MISSION CONTROL) にあるボタンの中から、目的に合ったものをクリックします。
    *   すると、AIへの「命令書（プロンプト）」がクリップボードにコピーされます。

2.  **【STEP 2: 目的の注入】**
    *   コピーした命令書を、お使いのAI（ChatGPT, Geminiなど）に貼り付けます。
    *   **【最重要】** 命令書の中にある **`【目的】：[入力]`** の部分に、あなたが「作りたいもの」を具体的に書き込んでください。（例：`【目的】：タイマーアプリ`）
    *   目的を空欄のまま実行すると、AIは「LCPの見本」として基本システムを出力します。

3.  **【STEP 3: 現実への注入】**
    *   AIが、`mkdir` と `cat` で構成された「黒いコマンドの塊」を生成します。
    *   その塊をすべてコピーし、あなたのPCのターミナル（黒い画面）に貼り付けて `Enter` キーを押してください。
    *   一瞬で、あなたのPCに新しいソフトウェアが誕生します。

**「エディタはもう要らない。ただ、目的を叫び、コマンドを執行するだけ。」**
これがLCPの真髄です。

## 各クラッキング・ユニットの解説

### I. THE_ATOMIC_CRACKER
-   **機能**: 完全に自由な目的で、あなただけのオリジナルツールをクラックします。
-   **使い方**: `[入力]` 部分に「〇〇をするためのPythonスクリプト」や「〇〇を管理するWebアプリ」のように、自由に目的を記述してください。

### II. CAPITAL_CRACKER
-   **機能**: LCPグランプリのスポンサーを募るための、投資家向け資料を生成します。

### III. REALITY_CRACKER
-   **機能**: 複数のプロセスをバックグラウンドで同時に動かす、高度な自律エージェントを構築します。

### IV. CIVILIZATION_STARTER
-   **機能**: よく使われる定番ツール（ウェブサイト、ToDoアプリ、ゲームなど）の完成品を、ボタン一つで召喚します。`[目的]` の入力は不要で、そのままAIに投げるだけで機能します。
